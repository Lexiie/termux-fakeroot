Fakeroot script for termux
Beta version

How to install

1. Download deb file
2. Install with apt or dpkg command 

apt install ./fakeroot_1.0_aarch64.deb
or
dpkg install . /fakeroot_1.0_aarch64.deb


How to use

Just type fakeroot on terminal
